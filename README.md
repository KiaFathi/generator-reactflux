## Getting Started


### Install Yeoman

To install Yeoman:

```bash
npm install -g yo
```


### Install the Generator

To install generator-reactflux from npm, run:

```bash
npm install -g generator-reactflux
```

### Generate Your Application

Run Yeoman and select your generator of choice:

```bash
yo
```
